## 逐段拆解

### 开场
"五天时间五条视频，加分五万做了一百万播放量，人类最强编导到底是怎么做的？今天我自己来拆自己。"

三句话完成：成绩背书 + 身份强化 + 内容预告。"拆自己"这个表述制造了"内部视角"的稀缺感——不是拆别人，是拆自己。

### 第一步：用户吸引
明确用户画像（有涨粉转型和商业化需求的博主、想做IP的素人），选择"解决问题"而非"解决情绪"。关键决策：起号期做干货不做情绪内容。

### 第二步：平台
"做短视频的本质是做产品" → 考虑环境 → 平台在推 2 分钟以上深度内容 → 做深度口播干货。不抓眼球，要抓价值。

### 第三步：内容
"把那些需要付费的内容免费放出来，换取流量。" 起号有流量又想赚钱，不能什么都要。

### 第四步：IP
"IP 从 ID 就开始了" — 名字就是定位。视觉符号：蜘蛛侠头套 + 白板。

### 第五步：SOP
选题库 → 白板大纲（不要脚本）→ 手机+麦克风拍摄 → 即时剪辑（删气口、加倍速、配BGM、做字幕）→ 晚上7点发布。

## 编导手法分析

### 开头钩子
成绩数字前置（5天/5条/5万粉/百万播放），密度极高，每个数字都是锚点。

### 节奏控制
142 秒拆解 5 个步骤，每步约 25-30 秒。步骤之间没有过渡语，直接切换。这是"清单体"视频的节奏——观众可以截图每一步作为独立知识卡片。

### 信息结构
五步法结构清晰：用户 → 平台 → 内容 → IP → SOP。每一步都有"为什么"（逻辑）和"怎么做"（操作），但"为什么"占比更大，说明这期视频的策略是"讲底层逻辑"而非"教操作步骤"。

### 结尾设计
"现在北京时间晚上5点，我相信7点之前一定能看到这条视频。" — 用真实时间制造"即时感"和"效率感"，同时暗示发布时间策略（7点）。

## 底层逻辑提炼

1. **起号 = 解决问题 > 解决情绪** — 起号期选干货不选情绪，因为干货的可收藏性和可分享性更高。
2. **付费内容免费化 = 流量杠杆** — 用"别人收费的我免费"制造信息差，换取流量。
3. **IP 从 ID 开始** — 名字就是定位，视觉符号就是记忆点，不需要等内容积累。
4. **即时剪辑 = 心力密度** — 不拖延、不精修，用速度保证内容的"鲜活感"。
5. **发布时间 = 心理学** — 晚上7点因为"人吃饱了耐心更好"，不是拍脑袋。
